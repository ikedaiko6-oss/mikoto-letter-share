# ミコトからの手紙

AI占いミニWebアプリの静的プロトタイプです。

## 使い方

`index.html` をブラウザで開くだけで動きます。

ローカルサーバーで見る場合:

```bash
cd future-log-diagnosis
python -m http.server 8080
```

Windows/Macどちらでも動かしやすいように、現時点ではNode.js依存なしです。

## 現在の仕様

- 入力: ニックネーム、生まれ月、テーマ、悩み、今の状況
- 出力: 今日の一通、ミコトのひとこと、行動メモ、有料版「ミコトの七日便り」導線
- 結果画像の保存
- 有料レポート導線の仮置き
