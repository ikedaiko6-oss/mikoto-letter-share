# 「自分らしく生きたい」カテゴリ追加リサーチ依頼

## 依頼概要

「ミコトからの手紙」の診断カテゴリに、28個目として `自分らしく生きたい` を追加したいです。

実際に妻に試してもらったところ、「自分のこと」で占おうとしたが、一覧に近いものがなかったとのことでした。詳しく聞くと、欲しかった選択肢は「自分らしく生きたい」でした。

このカテゴリを、既存の27カテゴリと同じ品質で正式追加できるように、リサーチと文面案をお願いします。

## 追加したいカテゴリ

| key | 表示名 |
|-----|--------|
| `be-yourself` | 自分らしく生きたい |

## 既存カテゴリとの違いを整理したい

近いカテゴリはすでにありますが、ユーザー視点では別の入口に見えています。

| key | 表示名 | 方向性 |
|-----|--------|--------|
| `true-self` | 自分の本音がわからない | 自分の気持ちが見えない、迷子感 |
| `lost-self` | 自分を取り戻したい | 過去の自分・置き去りにした感覚を迎えに行く |
| `no-more-unwanted` | 嫌なことをしたくない | 境界線、拒否、もう自分を差し出したくない |
| `be-yourself` | 自分らしく生きたい | これからの生き方、役割や期待から少し自由になりたい |

## 調べてほしいこと

1. 「自分らしく生きたい」と相談する人の本音
2. その人が救われる言葉の型
3. ミコト文に入れる刺さる一文
4. 既存カテゴリとの差別化
5. `app.js` に反映しやすい文面案

## リサーチ観点

Yahoo!知恵袋などの相談文から、次のような本音が本当に出ているか見てください。

- 人に合わせすぎて、自分の人生じゃない気がする
- 妻、母、会社員、娘などの役割に埋もれている
- わがままになりたいわけではないが、もう少し自分で選びたい
- 自分らしさが何かは曖昧だが、今のままでは苦しい
- 周りから見た正解を選び続けることに疲れている
- 好きなもの、やりたいこと、嫌なことを後回しにしてきた

## ミコト文スタイル

- 断定しない
- 説教しない
- スピリチュアルに寄せすぎない
- 本音を先に言い当てる
- 占いっぽさは残す
- 最後は現実の小さな行動に落とす
- 「あなたは悪くない」だけで終わらせず、次の確認を出す
